# HireCraft AI // Optimized Resume Node

HireCraft AI is a premium, developer-facing resume optimization and refinement engine. It uses advanced LLMs to inject quantitative impact, harmonize terminology with target Job Descriptions, and provide real-time recruiter simulation.

## 🚀 Engine Initialization

Follow these steps to initialize the HireCraft environment locally:

### 1. Repository Setup
```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>
```

### 2. Dependency Installation
```sh
npm install
```

### 3. Local Execution
```sh
# Start the development server with instant preview
npm run dev
```

## 🛠️ Technology Suite

This project is built with high-fidelity technologies:

- **Vite**: Rapid engine initialization and HMR.
- **TypeScript**: Type-safe node processing.
- **React**: Functional component architecture.
- **shadcn-ui**: Sharp, high-density UI primitives.
- **Tailwind CSS**: Utility-first atomic styling.

## 🌐 Production Deployment

For production deployment, you can use **Vercel**, **Netlify**, or any modern static hosting platform.

1. Run the build command: `npm run build`
2. Deploy the generated `dist` folder.
